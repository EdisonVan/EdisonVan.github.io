Go 语言的优点

生产力语言

作为一门生产力语言，它有很多约束，规避代码的一些低级错误！
e.g..go 语言中没有前置的++和–，所以你不用担心混淆前置++和后置++的区别。

可读性好，易于维护

e.g.位运算符重的按位置零操作&^

  • 右边操作数位为 1 时，结果都为 0
  • 右边操作数位为 0 时，结果和左侧保持一直，左侧为 0，则结果为 0
1 &^ 0 -- 1
1 &^ 1 -- 0
0 &^ 1 -- 0
0 &^ 0 -- 0
